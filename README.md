These scripts queries the CMIP7 Data Request v1.2.2.2 for various data fields. 

Setting up environment to run the scripts: The following commands will need to be run to set up a virtual environment. 
    python -m venv data-request-cmip7
    source data-request-cmip7/bin/activate
    pip install --upgrade pip
    pip install CMIP7-data-request-api

Before running these scripts you will need to activate the virtual environment with the command: 
    source data-request-cmip7/bin/activate
